# **App Name**: ScamSlayer AI

## Core Features:

- Content Analysis: Analyze user-submitted text (SMS, email, social media posts, etc.) for potential scam indicators.
- Risk Assessment: Evaluate the likelihood of the content being fake or a scam, providing a probability percentage.
- Scam Type Identification: Classify the detected scam into categories like 'Phishing,' 'Financial Scam,' or 'Fake News.'
- Explanation Generation: Generate a simple explanation of why the content is risky or safe for the end user.
- Safety Advice: Provide actionable advice on what steps users should take (e.g., 'Do not click the link,' 'Verify from official website').
- Content Input: A text field for the user to paste the message, link or any other content.
- AI Reasoning Tool: The AI LLM uses reasoning as a tool to analyze an arbitrary message provided as input, to decide if and when to include a reason from a list of well-known deceptive manipulations.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire trust and security.
- Background color: Light blue-gray (#ECEFF1) for a clean, professional look.
- Accent color: Yellow-orange (#FFAB40) to draw attention to important warnings.
- Body and headline font: 'Inter', sans-serif, for clarity and readability.
- Use clear and recognizable icons to represent scam types and safety advice.
- Maintain a clean and straightforward layout that emphasizes important warnings.
- Use subtle animations to highlight risk levels.
